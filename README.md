# Censor
A simple and robust chat censoring plugin for Hytale

# Installation
Download the [latest avaliable release](https://github.com/sigreturn/censor/releases/latest) and put the ``censor-x.x.x.jar`` file into your server's ``mods`` folder and restart the server.

# ⚠️ Warning
At the moment, this plugin currently only functions if running on a Linux/amd64 machine. Other platforms will be addressed in future releases.
